# How to set up

1. Open Command line in the file that you downloaded all these files
2. Type `pip install virtualenv`
3. Type `virtualenv E`
4. Type `cd E/Scripts/`
5. Type `activate`
6. Type `pip install -r require.txt`


# How to run
1. Open Command line in the file that you downloaded all these files
2. Type `cd E/Scripts/`
3. Type `activate`
4. Type `python "<file_name.py>"` .........> Example : `python "See a monkey.py"`
